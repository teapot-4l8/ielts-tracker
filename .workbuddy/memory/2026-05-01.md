## IELTS Tracker 自动本地备份功能

### Book/Test 选项目持久化（Record Test）
将 `ScoreInputForm` 中的 `book`/`testNum` state 提升到 `App.jsx`（`selectedBook`/`selectedTestNum`），通过 props 传入。切换 tab 后选择不会重置。

### Study Log 书本次选择持久化（同上）
将 `StudyLog` 中的 `selectedBook`/`selectedTest` state 也提升到 `App.jsx`，通过 `ProgressDashboard` props 链式传给 `StudyLog`。
- 数据文件：`public/data/records.json`（随项目纳入 Git）
- Vite 插件（`vite.config.js` 的 `dataFilePlugin`）拦截 `/data/records.json` 的 GET/PUT 请求，直接读写文件系统
- 每次任意数据（records/studyProgress/dailyTodos）变化时，通过防抖（500ms）的 `triggerAutoSave()` 调用 `saveAll()` 写入文件
- 页面加载时，`App.jsx` 的 `useEffect` 调用 `fetchAll()` 读取文件，与 localStorage 对比后决定是否需要 reload 恢复
